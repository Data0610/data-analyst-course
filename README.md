# 🐍 Python & Data Wrangling: Guía de Fundamentos 

¡Bienvenido a mi repositorio público! / Welcome to my public repository!
Este repositorio contiene una guía estructurada y práctica para aprender los fundamentos de programación en **Python** y dominar las técnicas de limpieza, manipulación y análisis de datos (**Data Wrangling**) utilizando la poderosa librería **Pandas**.

## 📚 Estructura del Contenido

Dentro de cada carpeta de idioma, encontrarás la información dividida en tres módulos de aprendizaje progresivo:

### 🟢 Módulo 1: Python Básico (Vol. 1)
Aquí cubrimos los cimientos de la programación en Python.
* Introducción, Sintaxis y el comando `print`.
* Creación y reglas de Variables.
* Operadores Aritméticos.
* Manipulación profunda de Cadenas de Texto (Strings).

### 🟡 Módulo 2: Python Básico (Vol. 2)
Avanzamos hacia el control de flujo y las estructuras de datos.
* Colecciones: Listas y Diccionarios.
* Control de flujo: Sentencias Condicionales (`if`, `elif`, `else`) y Bucles.
* Modularidad: Creación de Funciones personalizadas y alcance de variables.

### 🔵 Módulo 3: Manipulación de Datos (Data Wrangling) (Vol. 1)
Damos el salto al análisis de datos corporativo con Pandas.
* Ingesta de datos: Lectura avanzada de archivos CSV y Excel.
* Análisis exploratorio: Visualización de estructuras y estadísticas resumidas.
* Limpieza de datos: Tratamiento de valores nulos (NaN), duplicados y formatos.
* Transformación: Técnicas de agrupamiento (`groupby`) usando Split-Apply-Combine.
* Filtrado avanzado: Indexación con `loc`/`iloc` y consultas complejas con `query()`.
